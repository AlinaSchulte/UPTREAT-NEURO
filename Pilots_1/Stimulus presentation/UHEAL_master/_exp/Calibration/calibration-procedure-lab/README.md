# Calibration Procedure for the lab #

### What is this repository for? ###

* Keeping up to date the calibration procedure for the headphones in the lab

### How do I get set up? ###

* Clone this repository
* That's it!

### Contribution guidelines ###

* Check the open issues, assign them to one of the contributors
* Add an issue if not present in the list already
* When working on an issue, create a new branch `iXX-short-name`, with XX the issue number. If this branch has multiple commits, try to clean it (`git rebase -i master`)before pushing it to the repository.
* When an issue is finished, create a pull request, to merge it in the master branch
